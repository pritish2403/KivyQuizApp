# Kivy Quiz App 🎮

A simple yet interactive multiple-choice quiz application built with **Python** and **Kivy**. Designed like a game show (inspired by *Kaun Banega Crorepati*), this app tests users with general knowledge questions and tracks their score with a money ladder!

---

## 🚀 Features

- 📚 Multiple categories of questions: GK, History, Geography, Science, Computer Science
- 🎨 Graphical UI using Kivy
- 💰 Money-based progression system
- ❌ Ends on wrong answer (like real quiz shows)
- 🧠 Educational and fun!

---

## 🛠️ Technologies Used

- **Python 3.13**
- **Kivy** for GUI
